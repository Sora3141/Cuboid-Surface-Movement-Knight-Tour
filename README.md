# KNIGHT TOUR // PRIME - Spatial Logic Engine

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Play%20Now-00f0ff?style=for-the-badge)](https://sora3141.github.io/Cuboid-Surface-Movement-Knight-Tour/)

## 概要
KNIGHT TOUR // PRIME は、直方体の表面上における「ナイトの巡歴問題」をインタラクティブにシミュレート・視覚化する3Dウェブアプリケーションです。
Three.jsを活用し、ガラスモーフィズムを取り入れたUIの中で、任意の次元におけるナイトの移動経路を探索することができます。

## 特徴
* **3D空間での巡歴シミュレーション**
  チェスのナイトの動きを直方体の展開図および表面に拡張しています。面と面をまたぐ移動（エッジの通過と座標系の回転）を計算し、3Dモデル上で正確に再現します。
* **動的なパラメータ変更**
  UIのスライダーを使用して、直方体の各辺の長さをリアルタイムに変更し、盤面を再構築できます。
* **視覚的なフィードバック**
  * 次に移動可能なマスをハイライト表示します。
  * 訪問済みの経路は軌跡として記録されます。
  * 全マス踏破時の演出や、手詰まり時の警告エフェクトを備えています。
* **クロスプラットフォーム対応**
  PCとスマートフォンの両方に最適化されており、モバイル端末向けには専用のHUDメニューとタッチ操作を提供します。

## 技術スタック
* **HTML5 / CSS3**: レスポンシブデザイン、CSSアニメーション
* **JavaScript**: モジュールベースの設計、面遷移のアルゴリズム実装
* **Three.js**: 3Dレンダリング、ライティング設定、カメラ操作

## 操作方法
1. **次元の設定**: コントロールパネルを開き、N、M、Lのスライダーで直方体のサイズを決定します。
2. **盤面の生成**: REBOOTボタンを押して新しい直方体モデルを生成します。
3. **ナイトの移動**: 光っているタイルを選択すると、ナイトがその位置に移動します。
4. **手戻し**: 間違えた場合はUNDOボタンで直前の状態に戻すことができます。
5. **目標**: 直方体の表面にあるすべてのタイルをちょうど1回ずつ訪問できればクリアです。画面をドラッグしてカメラを回し、裏面のタイルも確認しながら進めてください。

## ファイル構造
* `index.html` : メインの構造とUIレイアウト、Three.jsの読み込み
* `style.css` : UIデザイン、スタイリング、レスポンシブ設定
* `script.js` : 3D描画ロジック、ナイトの移動アルゴリズム、進行管理

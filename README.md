# KNIGHT TOUR // PRIME - Spatial Logic Engine

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Play%20Now-00f0ff?style=for-the-badge)](https://sora3141.github.io/Cuboid-Surface-Movement-Knight-Tour/)

## 📝 概要 (Overview)
**KNIGHT TOUR // PRIME** は、直方体（Cuboid）の表面上における「ナイトの巡歴（Knight's Tour）問題」をインタラクティブにシミュレート・視覚化する3Dウェブアプリケーションです。
Three.jsを活用し、サイバーパンクテイストの美しいGlassmorphism（ガラスモーフィズム）UIの中で、任意の次元 $(N \times M \times L)$ におけるナイトの移動経路を探索することができます。

## ✨ 特徴 (Features)
* **3D空間での巡歴シミュレーション**
  * チェスの「ナイト」の動きを直方体の展開図・表面に拡張。面と面をまたぐ移動（エッジの通過と座標系の回転）を正確に計算し、3Dモデル上で再現しています。
* **動的なパラメータ変更 (N × M × L)**
  * UIのスライダーを使用して、直方体の各辺の長さ（最大8×8×8）をリアルタイムに変更し、盤面を再構築（REBOOT）できます。
* **直感的な視覚エフェクト**
  * **ナビゲーション**: 次に移動可能なタイルがマゼンタ色でハイライトされます。
  * **トレース**: 訪問済みの経路はシアン色の軌跡として記録されます。
  * **フィードバック**: 全マス踏破時のパーティクルエクスプロージョン（成功）や、手詰まり時の画面揺れ・レッドフラッシュ（失敗）などの演出を備えています。
* **クロスプラットフォーム・レスポンシブ対応**
  * PCとスマートフォンの両方に最適化。モバイル端末向けには専用のHUD（ヘッドアップディスプレイ）メニューとタッチ操作を提供します。

## 🛠 技術スタック (Tech Stack)
* **HTML5 / CSS3**: レスポンシブデザイン、CSSアニメーション、Glassmorphism UI
* **JavaScript (ES6+)**: モジュールベースの設計、面遷移のアルゴリズム実装
* **Three.js (v0.160.0)**: 3Dレンダリング、ライティング設定、OrbitControlsによるカメラ操作

## 🎮 操作方法 (How to Use)
1. **次元の設定**: 左上のコントロールパネル（スマホの場合は画面右上の `◆` ボタンから展開）を開き、`N`, `M`, `L` のスライダーで直方体のサイズを決定します。
2. **盤面の生成**: `◆ REBOOT` ボタンを押して新しい直方体モデルを生成します。
3. **ナイトの移動**: マゼンタ色に光っているタイルをクリック（またはタップ）すると、ナイトがその位置に移動します。
4. **UNDO（一手戻す）**: 間違えた場合は `↺ UNDO` ボタンで直前の状態に戻すことができます。
5. **目標**: 直方体の表面にあるすべてのタイルをちょうど1回ずつ訪問できればミッションクリアです。画面をドラッグしてカメラを回し、裏面のタイルも確認しながら進めてください。

## 📁 構造 (Project Structure)
* `index.html` : メインの構造とUIレイアウト、Three.jsのインポートマップ
* `style.css` : UIデザイン、サイバーパンク風のスタイリング、レスポンシブ設定
* `script.js` : 3D描画ロジック、ナイトの移動アルゴリズム（面の展開・回転処理）、ゲームの進行管理

## 🌐 動作環境 (Environment)
最新のWebGL対応ブラウザ（Chrome, Firefox, Safari, Edgeなど）で動作します。
